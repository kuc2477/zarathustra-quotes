**이름높은 현자들에 대하여**

> 사막에는 예로부터 진실한 자, 자요로운 정신을 소유한 자들이 주인으로서 살아왔다. 반면 도시에는 자신들의 배를 넉넉히 채운 이름높은 현자들, 이를테면 민중이라는 이름의 수레나 끄는 짐승들이 살아왔다.  
>  
> *174p - 이름높은 현자들에 대하여*
